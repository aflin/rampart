Welcome to the Rampart Webserver

Your server is running, and can now be customized.

See the web_server/web_server_conf.js script in the Rampart installation directory. By default it can be found at /usr/local/rampart/web_server/web_server_conf.js but might vary depending on the options you chose upon installing.

There are also some examples included to get you started.

  • Show my request
  • Show my request (with highlighting)
  • A simple websockets chat app

There are a few more in the /usr/local/rampart/web_server/apps/test-modules/ directory.

Documentation for Rampart can be found here.

Enjoy!
